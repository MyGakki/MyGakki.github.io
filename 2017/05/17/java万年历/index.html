通过Calendar类实现万年历

功能为根据输入的年份输出该年日历，或者根据输入的日期判断周几


先上代码为敬。
代码部分并不难懂，我写的时候主要碰到三个问题。第一是Calendar类的使用，第二是Scanner类使用多个输入流出现的错误，第三个是SimpleDateFormat类。
首先是Calendar类，这个类是一个抽象类，所以也就意味着不能直接实例化，必须通过getInstance方法来获得一个对象。Calendar类中有几个重要的常量:
Calendar.MONTH是该Calendar对象中的月份，一月到十二月分别用int型的0，1，···，11表示。
Calendar.DAY_OF_MONTH指一个月中的某天，第一天是1。
Calendar。DAY_OF_WEEK是一星期中的某天，周日是1，周一是2···。
Calendar还有几个常用的方法:
set方法，set方法有多种重载，set(int filed, int value),第一个参数是要set对象，例如set(Calendar.Month,0)即是将月份设成1月，set(int year, int month, int date, int hourOfDay, int minute, int second),有前3个，5个和6个参数的重载。
setTime(Date date)使用给定的Date设置Calendar的值。
getTime()获得一个Date类。
add(int filed, int amount)根据日历的规则，为给定的日历字段添加或减去指定的时间量。

然后是一个比较蠢的问题，使用多个Scanner时，当关闭一个输入流之后，无法再打开，导致第二次会抛出java.util.NoSuchElementException错误。

SimpleDateFormat 是一个以与语言环境有关的方式来格式化和解析日期的具体类。它允许进行格式化（日期 -> 文本）、解析（文本 -> 日期）和规范化。SimpleDateFormat(String pattern)是一个构造方法，pattern是日期和时间格式的模式。例如本代码中的yyyy-mm-dd即是以年月日的格式。
